# 03-功能模块说明

## 用户功能

### 用户登录与注册

系统支持账号密码登录、注册、刷新登录状态和退出登录。登录成功后，用户可以进入工作台查看检测记录、报告和个人策略。

### URL 风险查询

用户可以在 Web 平台输入 URL 发起检测。系统返回风险等级、风险分数、命中规则、AI 分析状态、风险摘要和安全建议。

### 我的检测记录

用户可以查看自己的历史检测记录，包括 URL、域名、来源、风险等级、分数和检测时间。

### 报告详情

报告详情页展示检测结论、命中规则、行为信号、AI 分析结果、页面特征摘要、历史同域名记录和处置动作。

### 个人域名策略

用户可以维护个人信任域名、阻止域名和临时信任域名。策略会同步给浏览器插件，用于后续访问时快速判断。

### 插件绑定与同步

用户可以通过 Web 平台确认插件绑定请求，插件获取专用访问令牌后即可同步策略、上报事件和调用检测接口。

## 浏览器插件功能

### 自动检测

插件在网页加载完成后，根据本地配置判断是否自动检测当前页面。

### 页面特征采集

插件采集 URL、标题、可见文本、按钮文本、输入框提示、表单提交域名和是否存在密码输入框等信息。

### 风险提示与拦截

对于可疑或恶意网站，插件可以展示风险提示页，并提供返回安全页面、查看报告、临时信任或永久信任等操作。

### 本地策略缓存

插件缓存信任域名、阻止域名、临时信任记录和策略版本，减少重复请求，提高访问侧响应速度。

### 事件上报

插件可以将扫描、提醒、绕过、信任、反馈和错误事件上报后端，便于平台统计和复盘。

## 管理员功能

### 规则管理

管理员可以查看、创建、测试、更新和停用检测规则。规则用于提取网页行为信号并计算风险分数。

### 域名管理

管理员可以维护全局信任域名和全局阻止域名，对所有用户和插件生效。

### 样本与反馈管理

管理员可以查看用户反馈和待复核样本，对误报、漏报或风险确认进行处理。

### AI 配置管理

管理员可以配置 DeepSeek 或兼容服务的接入地址、模型名称和 API Key，并进行连通性测试。

### 插件管理

管理员可以查看插件默认配置、同步事件、插件实例和安全预警触发情况，并可撤销指定插件实例。

### 用户管理

管理员可以查看用户列表，创建用户，修改角色或状态，重置密码，禁用或启用账号。

### 统计分析

管理员可以查看检测总览、风险分布、来源分布、趋势统计、反馈趋势和插件事件统计。
