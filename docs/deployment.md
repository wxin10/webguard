# 部署文档

## 部署拓扑

### 本地开发环境

```
┌─────────────────┐     ┌─────────────────┐
│   浏览器插件      │     │   Web 前端      │
└────────┬────────┘     └────────┬────────┘
         │                       │
┌────────┴────────┐     ┌────────┴────────┐
│   反向代理 (Nginx)  │<───>│   后端服务 (FastAPI) │
└─────────────────┘     └────────┬────────┘
                                 │
                         ┌───────┴───────┐
                         │   数据库      │
                         └───────────────┘
```

## 前后端分层

### 前端层

- **静态资源**：由 Nginx 提供
- **API 代理**：由 Nginx 反向代理到后端
- **路由**：前端单页应用路由

### 后端层

- **API 接口**：FastAPI 提供 RESTful API
- **规则引擎**：处理规则匹配和评分
- **模型推理**：本地模型推理
- **数据存储**：PostgreSQL 数据库

## PostgreSQL 生产建议

### 配置优化

```
# postgresql.conf
max_connections = 100
shared_buffers = 256MB
effective_cache_size = 768MB
work_mem = 4MB
maintenance_work_mem = 64MB
min_wal_size = 80MB
max_wal_size = 1GB
checkpoint_completion_target = 0.9
random_page_cost = 4
effective_io_concurrency = 200
```

## HTTPS 与反向代理建议

### HTTPS 配置

- 使用 Let's Encrypt 或其他 CA 颁发的证书
- 配置 TLS 1.2+ 协议
- 启用 HSTS
- 配置 OCSP 装订

### 反向代理配置

- 配置负载均衡
- 配置健康检查
- 配置限流
- 配置缓存
